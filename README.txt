--- Simple Python Calculator

This project contains a simple calculator built in Python using Tkinter.

--- Files:
- **calculator.py**: Python script for the calculator.
- **calculator.exe**: Executable file for running the calculator without needing Python installed.

--- How to Use:
1. If you have Python installed, you can run the **calculator.py** script directly.
2. If you don't have Python installed, you can run the **calculator.exe** file to use the calculator without any additional setup.
